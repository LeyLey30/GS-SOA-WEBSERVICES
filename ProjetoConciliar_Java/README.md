Projeto: Conciliar Híbrido (Java - Spring Boot)

Conteúdo do ZIP:
- backend (Spring Boot application)
- docker-compose.yml (MySQL)
- tools/k6_load_test.js (load test)

Instruções rápidas:
1. docker-compose up -d
2. cd backend
3. ./mvnw spring-boot:run   (ou mvn spring-boot:run)
4. Acesse http://localhost:8080/swagger-ui.html
